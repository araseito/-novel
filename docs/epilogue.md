# エピローグ：物語は、続く


――第8章、完。


――第2部、完。


世界が、止まった。


女神ルミナスが、現れた。


そして――


真実が、明かされた。


「あなた、死んでないわよ」


転生ではなく――


**時間差転移。**


地球の一時間が――


異世界の一年。


三年間の冒険は――


地球では、三時間。


しのぶは、まだ生きている。


ただ、病院で眠っているだけ。


ルミナスの正体――


**地球出身の元人間。**


**サブカルオタク。**


**しのぶのガチファン。**


すべては――


**善意の演出**だった。


しのぶは、選んだ。


「一度、帰ります」


「でも、また来ます」


「今度は、私が選んで」


逃げるのではなく――


**選んで、戻る。**


それが――


しのぶの、答え。


光に包まれ――


しのぶは、地球へ。


残された弟子たちは――


誓った。


「師匠が戻るまで――」


「もっと、強くなろう」


「黒薔薇の園を、守ろう」


そして――


セラが、笑った。


「師匠が戻ってきた時――」


「どれだけ伝説が増えたか――」


「楽しみだな」


これは、終わりではない。


新しい、始まり。


地球と異世界を――


**行き来する物語**の――


序章。


師匠は、帰ってくる。


今度は――


**役を降りた状態**で。


**自分の意志**で。


そして――


また、面白いことが起きる。


また、勘違いが生まれる。


また、伝説が増える。


それを――


みんなが、待っている。


弟子たちも。


ルミナスも。


そして――


**地球でのガチファンサイン会**も。


「やったー！　約束だからね！」


ルミナスの、最後の叫び。


それが――


**第3部**への、予告。


現世と異世界を行き来する――


新しい、日常が――


始まる。


――To be continued...
