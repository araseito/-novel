# 第1章：クラン《黒薔薇の園》爆誕

## 第1節：掲示板の前で


僕の名前は、ルーク・アッシュフォード。

18歳。冒険者ランクF。

つまり――最底辺の新人冒険者だ。


聖王都ルミナスの冒険者ギルド本部。

その一階、掲示板の前で、僕は立ち尽くしていた。


掲示板には、無数の依頼が貼られている。

だが、そのほとんどに――


**《受注不可》**


赤い札が貼られていた。


「今日も、全滅か……」


隣に立っていた冒険者が、苦い声で呟く。

見れば、Dランクの冒険者証を首から下げた、歴戦の風格を持つ男だった。


「あの、すみません……どういうことですか？」


僕は、思わず声をかけていた。


男は、僕の首元――Fランクの証――を一瞥し、ため息をついた。


「新人か。なら教えといてやる」


彼は、掲示板を指さした。


「ここ最近、まともな依頼は全部《黒薔薇の園》が持っていく。俺たちに残るのは、ゴブリン退治か薬草採取ぐらいだ」


「黒薔薇の……園？」


「知らねえのか？　《黒薔薇の園》。聖王都ルミナス最強のクランだ」


最強――


その言葉に、僕の胸が高鳴った。


「どれくらい、強いんですか？」


「どれくらい、ねえ……」


男は、顎に手を当てて考え込む。


「クランマスターのシルヴィア・ド・ノワール。噂じゃ、聖典の化身とも言われてる」


「聖典の……化身？」


「ああ。『気高き笑みが野に薔薇を咲かせ、天を清める』――聖典の一節そのものを体現してるって話だ」


男の声には、畏敬と諦めが混じっていた。


「それに、副長のエリザベート・フォン・クリムゾン。剣聖だ。Aランク。一太刀で魔獣の群れを薙ぎ払う」


「Aランク……！」


僕は、息を呑んだ。

冒険者ランクは、F・E・D・C・B・A・Sの七段階。

その上から二番目――Aランクは、国家に名を知られる英雄クラスだ。


「もう一人、アマンダ・"マンディ"・ストーンハート。拳闘王。こっちもAランク。素手で岩竜を殴り倒したって伝説がある」


「岩竜を……素手で……？」


「ああ。まあ、噂は半分に聞いとけ。だが、実力は本物だ」


男は、掲示板に目を戻した。


「そして今朝も、また一つ――大型依頼が消えた」


彼が指さしたのは、掲示板の最上段。

そこには、《受注済》の札が貼られた跡が残っていた。


「《森林地帯の魔獣群討伐》。報酬金貨五十枚。普通なら、Bランク三人編成で挑む案件だ」


「それを……」


「ああ。《黒薔薇の園》が、たった三人で受けた。しかも、昨日受注して今日にはもう出発済みだ」


信じられない、という表情を僕が浮かべると、男は肩をすくめた。


「慣れろ。ここじゃ日常だ」


そう言い残して、男は掲示板から離れていった。




僕は、再び掲示板を見上げた。


無数の依頼。

そして、無数の《受注不可》。


（これじゃあ、僕は一生Fランクのままだ……）


新人冒険者の壁は、厚い。

依頼は取れない。経験は積めない。ランクも上がらない。


この悪循環を、どうやって抜け出せばいいのか――


その時だった。


「おい、見ろよ！　新しい張り紙だ！」


誰かの声に、掲示板前の冒険者たちがざわめいた。


僕も、視線を巡らせる。


そして――見つけた。


掲示板の隅、少し目立たない位置に貼られた一枚の紙。


**《黒薔薇の園》新規メンバー募集**

**条件：やる気のある者**

**ランク不問**

**面接は本日夕刻、ギルド三階応接間にて**


「……え？」


僕は、思わず声を漏らした。


「ランク、不問……？」


周囲の冒険者たちも、ざわついている。


「マジかよ……黒薔薇が、メンバー募集？」

「しかもランク不問だって？　罠だろ、これ」

「いや、でも署名がある。本物じゃねえか？」


冒険者たちの会話が、耳に流れ込んでくる。


僕は、紙をもう一度見つめた。


**条件：やる気のある者**


やる気――


それなら、僕にもある。


冒険者になると決めた時から、僕は本気だった。

故郷を出て、聖王都ルミナスまで来た。

Fランクでも、必ず這い上がってみせると、そう誓った。


だが――


（本当に、僕みたいな新人でもいいのか……？）


迷いが、胸をよぎる。


その時、掲示板の向こう――ギルドの階段を、一人の女性が降りてきた。


赤い髪。

鋭い瞳。

腰には、美しい装飾の施された長剣。


そして、首元には――Aランクの冒険者証。


「エリザベート・フォン・クリムゾン……！」


誰かが、小さく叫んだ。


周囲の冒険者たちが、道を開ける。


彼女――リザは、まっすぐに掲示板へと歩み寄り、張り紙の前で立ち止まった。


そして、一言。


「面接希望者は、遠慮なく来なさい」


凛とした声。

だが、どこか温かみがあった。


「我がクランは、強さではなく、心を見る。師匠の教えに従い、志ある者を歓迎する」


師匠――


それは、クランマスター、シルヴィア・ド・ノワールのことだろう。


リザは、もう一度掲示板を見回し、そして階段を上っていった。




静寂が、一瞬だけ訪れた。


そして――


「よし、俺も行くぞ！」

「待て待て、準備が……！」

「チャンスじゃねえか、これ！」


冒険者たちが、一斉に動き出した。


僕も――


気づけば、階段を駆け上がっていた。


心臓が、激しく鳴っている。


（これは、チャンスだ）


いや――


（これしか、ない）


Fランクの僕が、這い上がるための唯一の道。


それが、目の前に現れたのだ。




三階。

応接間の前には、すでに長蛇の列ができていた。


冒険者たち――DランクやCランクも混じっている。

皆、緊張した面持ちで、順番を待っていた。


僕も、列の最後尾に並んだ。


そして――


扉の向こうから、声が聞こえてきた。


「おーっほほほほほっ！」


高笑い。

優雅で、気高く、そして圧倒的な存在感を持った笑い声。


周囲の冒険者たちが、一斉に息を呑む。


「あれが……シルヴィア・ド・ノワールの笑み……」

「聖典の一節、そのものだ……」


僕も、思わず立ち尽くした。


（すごい……）


まだ姿も見ていないのに、ただ笑い声だけで――


この場の全員が、跪きたくなるような、そんな威圧感。


いや、威圧感ではない。


これは――


**畏敬**だ。


僕は、拳を握りしめた。


（絶対に、面接を通してみせる）


そして――


この伝説のクランの一員になるんだ。




列は、ゆっくりと進んでいく。

一人、また一人と、応接間に入っていく。


そして――


僕の番が、やってきた。


「次、どうぞ」


リザの声が、扉の向こうから聞こえた。


僕は、深呼吸をして――


扉を開けた。




応接間は、想像以上に豪華だった。

壁一面の薔薇の紋章。窓からは王宮が見える。


そして――


中央のソファに、一人の女性が座っていた。


黒い髪。

紫の瞳。

優雅なドレス。


そして、その佇まいからは――まるで貴族のような、いや、それ以上の気品が溢れていた。


「ようこそ、《黒薔薇の園》へ」


彼女が、微笑んだ。


「私が、クランマスターのシルヴィア・ド・ノワール。あなたの名前は？」


「ル、ルーク・アッシュフォードです！」


僕は、必死に声を絞り出した。


「ランクは？」


「F、Fランクです……！」


正直に答えると、リザとマンディ――応接間の両脇に立っていた二人――が、一瞬だけ視線を交わした。


だが、シルヴィアは、表情を変えなかった。


「やる気は、あるかしら？」


「あります！　絶対に、強くなりたいんです！」


僕は、まっすぐに彼女の瞳を見つめた。


「冒険者として――誰かを守れるような、そんな存在になりたいんです！」


沈黙。


シルヴィアは、僕をじっと見つめていた。


そして――


「……いいでしょう」


彼女が、微笑んだ。


「あなたを、《黒薔薇の園》の一員として迎えますわ」


「え……本当ですか！？」


「ええ。志ある者は、私たちの仲間ですもの」


リザとマンディが、頷く。


「師匠の目に狂いはありません」

「よろしくな、ルーク！」


僕は――


信じられなかった。


本当に、受け入れてもらえたのだ。


「ありがとうございます！　絶対に、期待に応えます！」


僕は、深く頭を下げた。


そして――


この瞬間、僕の運命が変わった。


いや――


僕だけではなく、この聖王都ルミナス全体を巻き込む、巨大な勘違いの螺旋が――


今、まさに始まろうとしていた。
