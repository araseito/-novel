# 第4章：弟子たちの試練と覚醒

## Episode 1：別れと旅立ち


（しのぶ視点）


S級冒険者認定から、三ヶ月が経った。


その間――


私たちは、いくつもの依頼をこなしてきた。


魔獣討伐、遺跡探索、護衛任務。


どれも――


無事に、完了した。


（というか……）


私は、執務室で書類を見ながら思う。


（弟子たちが、強すぎる……）


リザ、マンディ、セラ。


三人とも――


最初に出会った頃とは、別人のように強くなっていた。


特に――


戦闘中の判断力と、連携が素晴らしい。


（私が何もしなくても……）


（勝手に、勝ってしまう）


それは――


嬉しいことでもあり。


少し、寂しいことでもあった。


＊　　＊　　＊


その日の夕方。


私は、三人を執務室に呼んだ。


「師匠、お呼びでしょうか？」


リザが、礼儀正しく訊ねる。


「ええ」


私は、三人を見る。


「少し、話があるの」


「話……ですか？」


マンディが、首を傾げる。


「ええ」


私は、深く息を吸った。


（これは――）


（私が、決めたこと）


「あなたたち――」


私は、真剣に言う。


「もう――」


「私から、学ぶことは――」


「ないわ」


「！」


三人が、驚いた顔をする。


「し、師匠！？」


リザが、慌てる。


「何を言ってるんですか！」


「まだまだ、学ぶことだらけです！」


「そうだよ、師匠！」


マンディも、焦った様子で言う。


「僕たち、まだ全然――」


「いいえ」


私は、首を振った。


「あなたたちは――」


「もう、十分強い」


「戦術も、連携も――」


「完璧よ」


「……」


三人が、黙り込む。


「でも――」


私は、続ける。


「それは――」


「**私の型**を、学んだだけ」


「！」


「あなたたちには――」


私は、一人ずつ見る。


「**あなたたち自身の型**が、必要なの」


「自分の……型……？」


セラが、訊ねる。


「ええ」


私は、頷く。


「私が教えられるのは――」


「基礎と、考え方だけ」


「でも――」


「**本当の強さ**は――」


「あなたたち自身が、見つけるもの」


「……」


三人が、真剣な顔になる。


「だから――」


私は、提案する。


「一度――」


「**別々に、修行**しなさい」


「別々に……！？」


リザが、叫ぶ。


「それって――」


「師匠と、離れるってことですか！？」


「ええ」


私は、静かに言う。


「三ヶ月――」


「いいえ、半年」


「半年間――」


「それぞれが、別の場所で――」


「**自分の強さ**を、見つけなさい」


「……」


部屋が、静まり返った。


「師匠……」


マンディが、震える声で言う。


「それって――」


「僕たちが、弱いから……？」


「違うわ」


私は、微笑む。


「あなたたちが、強いから」


「強すぎるから――」


「今のままでは――」


「**止まってしまう**の」


「！」


「もっと――」


私は、力強く言う。


「もっと、強くなれる」


「もっと、輝ける」


「そのためには――」


「一度、離れる必要がある」


「……」


三人が、顔を見合わせる。


そして――


リザが、一歩前に出た。


「師匠」


リザが、真剣に言う。


「わかりました」


「私――」


「自分の強さを、見つけてきます」


「リザ……」


「そして――」


リザが、微笑む。


「半年後――」


「もっと強くなって――」


「戻ってきます」


「師匠のために」


「……ありがとう」


私は、頷いた。


「僕も！」


マンディが、拳を握る。


「僕も、行く！」


「自分の強さを――」


「見つけてくる！」


「マンディ……」


「私も……」


セラが、静かに言う。


「行きます」


「私にも――」


「見つけたいものが、ある」


「セラ……」


私は、三人を見た。


（みんな……）


（ありがとう）


＊　　＊　　＊


翌日。


私たちは、それぞれの修行先を決めた。


「私は――」


リザが、地図を指す。


「**王都闘技場**に、行きます」


「そこには――」


「**剣聖エリシア**がいると聞きました」


「彼女の下で――」


「剣を、極めます」


「剣聖エリシア……」


私は、頷く。


（聞いたことがある）


（伝説の女剣士――）


（確か、引退して闘技場の師範をしているはず）


「いい選択ね」


「はい！」


リザが、嬉しそうに笑う。


「僕は――」


マンディが、別の場所を指す。


「**ダークフォレスト・ダンジョン**に、行く」


「そこには――」


「**ドワーフの鍛冶師ギムリ**がいるらしい」


「彼から――」


「武器の扱いと、鍛え方を学ぶ」


「ドワーフの鍛冶師……」


私は、感心する。


（武器を知ることは――）


（戦いを知ること）


（いい選択だわ）


「頑張ってね」


「うん！」


マンディが、力強く頷く。


「私は――」


セラが、最後に言う。


「**天空試練の塔**に、行きます」


「そこには――」


「**魔導師アルカナ**がいると聞きました」


「彼女から――」


「魔法と、戦術を学びます」


「試練の塔……」


私は、驚く。


（あそこは――）


（かなり、危険な場所のはず）


（でも――）


（セラなら、大丈夫かもしれない）


「気をつけてね」


「はい」


セラが、微笑む。


＊　　＊　　＊


そして――


別れの日が来た。


朝。


ギルドの前。


私と三人が、並んで立っている。


「じゃあ――」


私は、三人を見る。


「半年後――」


「ここで、会いましょう」


「はい！」


三人が、声を揃える。


「師匠――」


リザが、涙目になる。


「寂しいです……」


「私も」


私は、微笑む。


「でも――」


「これは、必要なこと」


「はい……」


リザが、涙を拭う。


「師匠――」


マンディが、拳を差し出す。


「約束だよ」


「半年後――」


「絶対、会おうね」


「ええ」


私は、彼の拳に、自分の拳を合わせる。


「約束よ」


「師匠……」


セラが、静かに言う。


「ありがとうございました」


「あなたに出会えて――」


「私は、変われました」


「セラ……」


私は、彼の頭を撫でる。


「あなたも、ありがとう」


「……」


セラが、目を閉じる。


そして――


三人が、それぞれの方向へ歩き出した。


リザは、東へ。


マンディは、南へ。


セラは、北へ。


私は――


三人の背中を、見送った。


（みんな――）


（強くなってね）


（そして――）


（必ず、戻ってきて）


風が――


吹いた。


三人の背中が――


少しずつ、小さくなっていく。


私は――


最後まで、見送った。


＊　　＊　　＊


（ルーク視点）


僕は――


その光景を、遠くから見ていた。


師匠が――


三人の弟子を、送り出す姿を。


「すごいな……」


僕は、呟く。


「あれが――」


「本当の師匠なんだ」


弟子を、手放すことができる。


それは――


本当に、弟子を信じているから。


「僕も――」


僕は、決めた。


「強くならなきゃ」


「師匠みたいに――」


「誰かを、導けるように」


そう思いながら――


僕は、訓練場へ向かった。


＊　　＊　　＊


――Episode 1、完。


リザは、東の王都闘技場へ。


剣聖エリシアの下で、剣を極める。


マンディは、南のダークフォレスト・ダンジョンへ。


ドワーフの鍛冶師ギムリから、武器の真髄を学ぶ。


セラは、北の天空試練の塔へ。


魔導師アルカナから、魔法と戦術を学ぶ。


それぞれの道を――


それぞれの試練を――


彼らは、歩み始めた。


そして――


半年後。


彼らが手にするのは――


**ルミナスの加護**。


**異常な強さ**。


そして――


**守るべきもの**への、覚悟。


物語は――


三つに、分かれる。


――Episode 2へ続く。
