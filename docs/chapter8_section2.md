# 第8章：教会介入・枢機卿登場（Part2終幕）

## 第2節：「聖典の具現」として公式認定される危機


（しのぶ視点）


聖女認定式まで――


**あと三日。**


私の生活は――


さらに、過酷になった。


毎日、教会からの指導。


毎日、礼拝の練習。


毎日、説教の訓練。


「……」


私は、もう――


限界を超えていた。




「シルヴィア様」


枢機卿が、私に――


声をかける。


「はい……」


私は、力なく答える。


「明日は――」


枢機卿が、書類を見る。


「聖典の朗読練習です」


「聖典……」


「はい」


枢機卿が、頷く。


「あなたは、神の代行者」


「ですから――」


「聖典を、完璧に読めなければなりません」


「……」


私は、何も言えなかった。




「さらに――」


枢機卿が、続ける。


「認定式当日は――」


「**神託**を、授けていただきます」


「神託……？」


私は、驚く。


「はい」


枢機卿が、目を輝かせる。


「神の言葉を――」


「人々に、伝えていただきます」


「……」


（神託って……何を言えばいいの……）


私は、混乱した。




「ご安心ください」


枢機卿が、にこやかに言う。


「台本は、こちらで用意します」


「台本……」


「はい」


枢機卿が、別の書類を取り出す。


そこには――




**『神託（台本）』**


**「我が子よ、光の道を歩め」**


**「汝らの信仰、我は見ている」**


**「黒き薔薇は、光を守る剣なり」**




と、記されていた。


「……」


私は、呆然とした。


（これ……完全に、台本じゃない）


（神託じゃなくて、演技じゃない……）




「シルヴィア様？」


枢機卿が、不思議そうに訊ねる。


「何か、問題でも？」


「いえ……」


私は、首を横に振る。


「何も……」


「それは、良かった」


枢機卿が、微笑む。


「では、練習しましょう」


「……はい」




私は――


台本を、読み始めた。


「我が子よ、光の道を――」


だが――


その時。


また――


「おーっほっほっ……」


高笑いが、出た。


「！」


私は、慌てて止める。


「すみません……」


「いえいえ」


枢機卿が、感動した様子で言う。


「素晴らしい……」


「あの高笑い……」


「まさに、神の喜び……」


「……」


私は、絶望した。


（もう、何もかも……おかしい）




＊　　＊　　＊


認定式の前日。


私は――


大聖堂に、呼ばれた。


「シルヴィア様」


枢機卿が、私を案内する。


「こちらが――」


「明日、あなたが立つ場所です」


大聖堂の中央。


高い壇上。


そこには――


**金色の玉座**が、置かれていた。


「……」


私は、固まった。


「玉座……？」


「はい」


枢機卿が、頷く。


「聖女の座です」


「明日――」


枢機卿が、真剣に言う。


「あなたは、ここに座り――」


「人々に、神託を授けます」


「……」




枢機卿が、私を――


玉座に、案内する。


「どうぞ」


「……」


私は、ゆっくりと――


玉座に、座った。


冷たい。


硬い。


そして――


**重い。**


「素晴らしい……」


枢機卿が、感動する。


「まさに――」


「神の化身……」


「……」


私は、何も言えなかった。




その時――


大聖堂の扉が開いた。


「失礼します」


入ってきたのは――


**数十人の信者たち**だった。


「……？」


私は、不安になる。


「シルヴィア様」


枢機卿が、説明する。


「明日のリハーサルです」


「信者たちの前で――」


「神託を、授けてみてください」


「え……」


私が、戸惑っていると――


信者たちが――


一斉に、跪いた。


「「「聖女様！」」」


「……」


私は、固まった。




「さあ」


枢機卿が、促す。


「神託を」


「……」


私は、台本を見る。


手が、震えている。


（これ……本当にやるの？）


だが――


もう、逃げられない。


私は――


深呼吸して――


台本を、読み始めた。


「我が子よ――」


「光の道を、歩め――」




その瞬間――


信者たちが――


感動の声を、上げた。


「ああ……」


「神の声……」


「聖女様……」


「……」


私は、続ける。


「汝らの信仰――」


「我は、見ている――」


そして――


また。


「おーっほっほっ……」


高笑いが、出た。




だが――


信者たちは――


さらに、感動した。


「素晴らしい……」


「神の笑い声……」


「奇跡だ……」


拍手が、響く。


「……」


私は、完全に――


理解不能になった。


（これ……何……）


（私、何してるの……）




＊　　＊　　＊


リハーサルが終わった後。


私は――


控室に、戻った。


一人になる。


鏡を、見る。


そこには――


**白い法衣を纏った私**が、映っていた。


聖女の衣装。


金の刺繍。


ティアラ。


「……」


私は、鏡の中の自分を――


見つめた。


（これ……誰？）


声優の、しのぶ？


冒険者の、シルヴィア？


それとも――


**聖女**？


「……」


（分からない）


（もう、自分が誰だか――）


（分からない）




扉が、ノックされた。


「師匠」


ルークの声。


「入って……」


私は、力なく答える。


扉が、開く。


ルーク、リザ、マンディ、セラが――


入ってくる。


「……」


みんな、心配そうに――


私を、見ている。




「師匠」


ルークが、前に出る。


「大丈夫ですか？」


「……大丈夫じゃない」


私は、正直に答えた。


「もう――」


「自分が、誰だか分からない」


「師匠……」


「私は――」


私は、鏡を見る。


「しのぶなの？」


「シルヴィアなの？」


「それとも――」


「聖女なの？」


涙が、流れた。


「分からない……」


「もう、何も――」


「分からない……」




ルークが――


私の手を、握った。


「師匠」


「……」


「師匠は、師匠です」


ルークが、真剣に言う。


「しのぶでも、シルヴィアでも、聖女でもない」


「師匠は――」


「**師匠**です」


「ルーク……」


「僕たちにとって――」


ルークが、続ける。


「師匠は、ずっと――」


「師匠です」


「それだけは――」


「変わりません」


「……」


私は、涙が止まらなくなった。


「ありがとう……」


「ルーク……」




リザも――


私の手を、握る。


「師匠」


「私も――」


「同じです」


「師匠は、師匠」


「それ以外の何者でもありません」


マンディとセラも――


頷く。


「その通りですわ」


「僕たちは、ずっと――」


「師匠の味方です」


「……」


私は、みんなを見た。


そして――


小さく、微笑んだ。


「ありがとう……」


「みんな……」




だが――


その時。


また――


「おーっほっほっ……」


高笑いが、出た。


「！」


私は、慌てて止めようとする。


だが――


止まらない。


「おーっほっほっほっ……」


「師匠！」


ルークが、叫ぶ。


だが――


私は、笑い続けた。


「おーっほっほっほっほっ……」




そして――


ようやく、止まる。


「……」


私は、力なく――


ソファに、座り込んだ。


「もう、駄目……」


「笑いが、止まらない……」


「……」


みんなが、黙り込む。




その時――


ルークが――


決意した表情で、言った。


「師匠」


「一つ、提案があります」


「提案……？」


「はい」


ルークが、真剣に言う。


「師匠の演技を――」


「逆に、利用するんです」


「演技を……利用？」


私は、不思議に思う。


「どういうこと？」




ルークが――


説明し始めた。


「師匠は、元声優ですよね」


「……ええ」


「だったら――」


ルークが、続ける。


「**神託も、演技で乗り切れるはずです**」


「でも――」


私が、言いかけた時――


「いえ」


ルークが、首を横に振る。


「ただの演技じゃありません」


「師匠の演技で――」


「この状況を、**変える**んです」


「状況を……変える？」


「はい」


ルークが、目を輝かせた。


「詳しくは――」


「明日、説明します」


「でも、今は――」


ルークが、真剣に言う。


「師匠、信じてください」


「僕たちが――」


「必ず、師匠を――」


「**自由にします**」


「……」


私は、ルークを見つめた。


そして――


小さく、頷いた。


「分かった……」


「信じるわ、ルーク」
