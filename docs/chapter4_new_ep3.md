# 第4章：弟子たちの試練と覚醒

## Episode 3：マンディの試練


（マンディ視点）


ダークフォレスト・ダンジョンに到着したのは――


師匠と別れてから、五日後だった。

僕は、答える。

「え！？」


僕は、驚く。


「あなたが、ギムリ様！？」


「ああ」


ギムリが、頷く。

「ギムリ・アイアンハンマーだ」


「よろしく、お願いします！」


僕は、深く頭を下げた。


「ふむ……」

「礼儀は、悪くない」


「だが――」

「まずは――」


ギムリが、ダンジョンを指す。


「このダンジョンの――」


「**最深部**まで、行ってこい」


「最深部……！？」


僕は、驚く。


（ここは、確か――）


（A級ダンジョンのはず……）


「怖いか？」


ギムリが、訊ねる。


「いえ！」


僕は、首を振る。


「行きます！」


「師匠が――」


「見ていてくれるから！」


「師匠？」


「はい」


僕は、微笑む。

「素晴らしい師匠が、いるんです」


「ほう……」


ギムリが、興味深そうに頷く。

頼りだった。

僕は、剣を構える。


（魔獣が――）
